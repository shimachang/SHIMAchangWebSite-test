## 使用技術
* HTML
* SCSS
* JavaScript(es6)
* Webpack "^5.35.1"

## インストール
```
npm install
```

## 実行
* 開発用フォルダ作成＆ローカルサーバー立ち上げ
```
npm run dev
```
* 商業用フォルダ作成
```

npm run prod
```

* コード整形
```

npm run lint-fix
```
